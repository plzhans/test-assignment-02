=== Header

|===
| name | 설명

| `X-USER-ID"`
| 요청한 사용자의 식별값 (숫자)
| `X-ROOM-ID"`
| 요청한 사용자가 속한 대화방의 식별값 (문자)
|===

=== Response status

|===
| status | 설명

| `200`
| OK : 성공

| `403`
| BadRequest : 잘못된 요청

| `500`
| 서버 오류

|===
